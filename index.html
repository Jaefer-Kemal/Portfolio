About Me

I'm a graduate of the ALX Backend Web Development Specialization and a proud member of A2SV (Africa to Silicon Valley), where I actively solve Data Structures and Algorithms problems on LeetCode and Codeforces.

My expertise lies in backend development using Python, Django, and Django REST Framework, paired with PostgreSQL for database management. I also build frontend interfaces with React and Tailwind CSS to create complete, full-stack applications.

Currently, I'm exploring Machine Learning using TensorFlow and Scikit-learn, while also expanding my programming language repertoire with C# and Go.

I'm passionate about system integration and building real-world projects that solve practical problems.
